# Simple Interest Calculator

This is a simple interest calculator project. 

## Formula
Simple Interest = (Principal * Rate * Time) / 100

# GitHub Final Project

This repository is created as part of the "Introduction to Git and GitHub" final project. It demonstrates fundamental Git workflows, including branching, merging, and remote repository management.

## Project Contents
- **LICENSE**: Apache License 2.0.
- **simple-interest.sh**: A bash script designed to calculate simple interest based on principal, rate, and time inputs.
- **Documentation**: Includes CONTRIBUTING.md and CODE_OF_CONDUCT.md to ensure professional standards and clear contribution guidelines.

## Usage
The `simple-interest.sh` script can be executed in a bash-compatible terminal to calculate interest values by providing the principal amount, annual rate of interest, and time period in years.
